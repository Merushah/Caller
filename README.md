# Caller

A lightweight autodialer tool that allows you to make **single** or **bulk calls** using an API or CSV file. It supports importing numbers, processing them one-by-one, and tracking call statuses.

## Features
- Single call API
- Bulk calling via CSV
- Call status logging
- `.env` based configuration

## Setup
git clone https://github.com/Merushah/Caller.git
cd Caller
npm install   # or pip install -r requirements.txt
